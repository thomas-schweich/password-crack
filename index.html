HASHING

$ Computers can easily convert a string of text (like a password) to an unintelligible sequence of letters and numbers using a hashing algorithm.

$ This website uses the popular MD5 algorithm.

$ What is not easy for computers is unhashing. Hashing is weird--it is considered "irreversible". The only way to figure out what was put into a hashing algorithm is to choose a string, try hashing it, and see if it comes out the same as your hash. If it did, then you've found what was put into the hashing algorithm.
